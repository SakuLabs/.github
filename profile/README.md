## What is Saku?

**Saku (策)** — Japanese for *plan*, *strategy*, or *policy* — transforms the school dashboard into a tactical command center. Built for students and educators who treat academic life like a campaign: every task a mission, every deadline an objective, every study group a squad.

Stop juggling sticky notes, scattered calendars, and group chats across five different apps. Saku unifies tasks, schedules, and collaboration into a single dark-themed interface designed for long study sessions and sharp focus.

## Core Modules

### Task Management
Create, prioritize, and track assignments across subjects. Assign tasks to collaborators, mark dependencies, and watch real-time statistics update as you grind through your workload.

### Scheduler & Calendar
Three views, one source of truth — month calendar for the long game, daily hour-by-hour slots for tactical execution, and a Gantt-style timeline for deadline visualization. Drag-and-drop rescheduling included.

### Study Groups & Chat
Create rooms, invite classmates, collaborate in real time. Persistent message history, clean interface, zero friction.

## Tech Stack

`Next.js 16` · `React 19` · `TypeScript` · `shadcn/ui` · `Framer Motion` · `Tailwind CSS` · `SQLite3`

## Design Philosophy

- **Dark-first** — engineered for late-night study sessions
- **Tactical** — every screen optimized for decisive action
- **Responsive** — mobile, tablet, desktop
- **Animated** — smooth motion that aids comprehension, never distracts

## Get Started

Head to the [Saku repository](../../saku) to clone, install, and launch your command center.

```bash
pnpm install
pnpm run init-db
pnpm run seed-db    # optional demo data
pnpm run dev
```
